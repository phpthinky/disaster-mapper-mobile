# Disaster Mapper Mobile 🗺️

A mobile field reporting application for disaster risk management built with:

- Laravel 11
- NativePHP Air 3.0.4
- Inertia.js + React
- Tailwind CSS 4
- SQLite (offline support)

## Setup
```bash
composer install
npm install
php artisan native:install
php artisan native:jump
```

## About
Built for LGU field workers to report disaster incidents offline.
